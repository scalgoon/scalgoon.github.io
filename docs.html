Need some help?

Using these docs you can get necessary information on how to use my bots



How do i use commands?






To learn how to use commands like help do {prefix}help.





prefix varies based on setup settings


Why can some members do commands and i can't?






Some commands go off user permissions to work if you can't do some commands this means you lack the necessary permissions to use them.


To see what permissions are needed for that command do {prefix}help {command name}.




How do i set channels and roles?






To learn how to set channels and roles do {prefix}setup to see a list of what you can set.


Once you see what you want to set do {prefix}setup {option}. The same process applies to the remove command.







prefix varies based on setup settings


How do i change the bot's prefix?






To change the bot's prefix just do {prefix}prefix {what you want it to be here}.





prefix varies based on setup settings

max characters of prefix is 3
